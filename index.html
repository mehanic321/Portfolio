CMS Специализация

Экосистема WP

Мой основной инструмент для 80% задач.

  • Plugin Architecture: Разработка плагинов под бизнес-логику (Hooks, Actions, Filters). Работа с wpdb.
  • WooCommerce Core: Кастомизация чекаута, интеграция платежных шлюзов, работа с заказами через API.
  • Performance: Оптимизация Query Monitor, настройка Object Cache (Redis), отключение ненужного функционала.

1C-Bitrix

Enterprise решения и сложные интеграции.

  • Ядро D7: Использование современного ORM и API Битрикс вместо старых методов CIBlockElement.
  • Компоненты: Написание собственных компонентов и шаблонов (result_modifier.php, component_epilog.php).
  • Интеграции: Настройка обмена с 1С:Предприятие (товары, заказы, контрагенты). Композитный режим.

OpenCart / MVC

Легковесные магазины со строгой структурой.

  • MVC Architecture: Четкое разделение логики (Controller), данных (Model) и вывода (View/Twig).
  • Модификаторы: Работа с OCMOD/VQMOD для изменения ядра без правки исходников.
  • Migations: Опыт переноса магазинов с других движков, импорт/экспорт миллионов товаров.

PHP 8+

MySQL

Frontend
